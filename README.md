# claude-bc-skills

A collection of Claude Code skills for Business Central AL development.

## Skills

### `al-test`

Generates AL test codeunits for Business Central extensions. Given a description of functionality to test, it explores your project's existing test codeunits, determines the right feature area, and creates or extends test codeunits following standard BC AL test conventions.

## Installation

Install individual skills using the `npx skills add` command:

```bash
npx skills add https://github.com/davidpracilio/claude-bc-skills/tree/main/skills/al-test
```
